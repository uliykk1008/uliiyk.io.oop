Materi Class dan Object (Car)

Penjelasan

Class adalah blueprint untuk membuat objek. Pada contoh ini dibuat class Car dengan atribut model dan color serta method start().

Source Code

class Car {
  String model;
  String color;

  Car(this.model, this.color);

  void start() {
    print("Mobil $model warna $color dinyalakan.");
  }
}

void main() {
  Car mobil = Car("Sedan", "Hitam");
  mobil.start();
}

Run Program

Kesimpulan

Class digunakan untuk membentuk objek sehingga program lebih terstruktur dan mudah dikembangkan.
