Materi Class Product

Penjelasan

Penggunaan final dan named parameter constructor membuat kode lebih aman dan fleksibel.

Source Code

class Produk {
  final String id;
  final String nama;
  final double harga;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
  });

  @override
  String toString() =>
      "Produk(id: $id, nama: $nama, harga: $harga)";
}

void main() {
  Produk laptop = Produk(
    id: "P001",
    nama: "Laptop Gaming",
    harga: 15000000,
  );

  print(laptop);
}

Run Program

Kesimpulan

Named parameter dan final membantu menjaga keamanan data objek dan meningkatkan kualitas kode.
