1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek. Objek memiliki atribut (data) dan method (perilaku).

2. Konsep Dasar

  • Kelas adalah blueprint atau cetakan objek.
  • Objek adalah hasil dari kelas.
  • Constructor digunakan untuk menginisialisasi data.

3. Implementasi Kode Dart

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus berada dalam rentang 0.0 - 4.0');
    }
  }

  void tampilkanData() {
    print('===== DATA MAHASISWA =====');
    print('Nama  : $nama');
    print('NIM   : $nim');
    print('IPK   : ${ipk.toStringAsFixed(2)}');
    print('Predikat : ${cekPredikat()}');
    print('==========================');
  }

  String cekPredikat() {
    if (ipk >= 3.75) {
      return "Cumlaude";
    } else if (ipk >= 3.50) {
      return "Sangat Memuaskan";
    } else if (ipk >= 3.00) {
      return "Memuaskan";
    } else {
      return "Cukup";
    }
  }

  @override
  String toString() {
    return 'Mahasiswa(nama: $nama, nim: $nim, ipk: ${ipk.toStringAsFixed(2)})';
  }
}

void main() {
  Mahasiswa mhs = Mahasiswa('Uliya Gumpari', '251410008', 3.87);
  mhs.tampilkanData();
  print(mhs);
}

4. Kesimpulan

Dengan menggunakan konsep OOP di Dart, program menjadi lebih terstruktur, mudah dikembangkan, dan lebih rapi dalam pengelolaan data.
