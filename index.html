1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Objek dibuat berdasarkan class yang memiliki atribut dan method.

2. Konsep Dasar OOP

Konsep utama OOP meliputi: - Class - Object - Encapsulation - Inheritance - Polymorphism

3. Class Sederhana (Car)

Class adalah blueprint untuk membuat objek. 

class Car {
  String model;
  String color;

  Car(this.model, this.color);

  void start() {
    print("Mobil dinyalakan");
  }
}

4. Object Instantiation

Object dibuat dari class menggunakan constructor. 

void main() {
  Car mobil = Car("Sedan", "Hitam");
  mobil.start();
}

5. Constructor

Constructor digunakan untuk menginisialisasi nilai awal objek. 

Car(this.model, this.color);

6. Practice: Class Mahasiswa

Contoh implementasi class Mahasiswa dengan atribut dan method. 

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk);
}

7. Encapsulation

Encapsulation adalah pembungkusan data dengan cara membuat atribut menjadi private dan menggunakan getter/setter.

8. This Keyword

Keyword this digunakan untuk merujuk ke atribut milik class.

9. Product Class

Contoh penggunaan named parameter dan final.

10. Kesimpulan

OOP membantu membuat program lebih terstruktur, mudah dipelihara, dan scalable melalui penggunaan class dan object.

11. Quiz Bonus

Buatlah class Buku dengan atribut judul dan penulis, serta method tampilkan().
