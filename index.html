1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Dalam OOP, program disusun menggunakan class dan object sehingga lebih terstruktur, mudah dikembangkan, dan mudah dipelihara.

2. Konsep Dasar OOP

Konsep utama dalam OOP meliputi:

  • Class
  • Object
  • Encapsulation
  • Inheritance
  • Polymorphism

3. Class Sederhana (Car)

Class merupakan blueprint untuk membuat objek. Pada contoh ini dibuat class Car.

4. Object Instantiation

Object dibuat dari class menggunakan constructor.

5. Constructor

Constructor digunakan untuk menginisialisasi nilai awal dari suatu objek ketika dibuat.

6. Practice Class Mahasiswa

Contoh implementasi class Mahasiswa dengan atribut, constructor, dan method.

7. Encapsulation

Encapsulation adalah proses pembungkusan data dengan membuat atribut menjadi private dan mengaksesnya menggunakan getter dan setter.

8. This Keyword

Keyword this digunakan untuk merujuk pada atribut yang dimiliki oleh class.

9. Product Class

Contoh penggunaan final, required, dan named parameter dalam constructor.

10. Kesimpulan

Dari materi yang telah dipelajari dapat disimpulkan bahwa OOP membantu membuat program lebih terstruktur dan modular melalui penggunaan class dan object. Konsep encapsulation, constructor, dan penggunaan keyword seperti this membuat kode lebih aman dan terorganisir.

11. Quiz Bonus
