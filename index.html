OBJECT ORIENTED PROGRAMMING

Uliya Gumpari | 251410008

1. Pengertian OOP

Object Oriented Programming adalah paradigma pemrograman yang berorientasi pada objek. Pendekatan ini membuat program lebih terstruktur, modular, serta mudah dikembangkan karena setiap bagian program dibangun dalam bentuk objek.

2. Konsep Dasar OOP

Konsep utama OOP meliputi Class, Object, Encapsulation, Inheritance, dan Polymorphism. Konsep-konsep ini membantu dalam membangun sistem yang fleksibel, reusable, dan lebih mudah dipelihara.

3. Class Sederhana (Car)

Class adalah blueprint untuk membuat objek. Pada contoh ini, class Car memiliki atribut model dan color serta method start() untuk menjalankan aksi.

4. Object Instantiation

Object instantiation adalah proses membuat objek dari class. Objek dibuat menggunakan constructor dan dapat langsung digunakan.

5. Constructor

Constructor adalah method khusus yang otomatis dijalankan ketika objek dibuat untuk menginisialisasi nilai atribut.

6. Practice Class Mahasiswa

Implementasi class Mahasiswa dengan atribut nama, nim, dan ipk. Terdapat method untuk menampilkan data serta menentukan predikat.

7. Encapsulation

Encapsulation adalah teknik menyembunyikan data dengan membuat atribut private dan mengaksesnya melalui getter dan setter untuk menjaga keamanan data.

8. This Keyword

Keyword this digunakan untuk merujuk pada atribut milik class ketika terjadi kesamaan nama antara parameter dan atribut.

9. Product Class

Class Product menunjukkan penggunaan final untuk membuat atribut tidak dapat diubah serta penggunaan named parameter agar lebih fleksibel.

10. Kesimpulan

OOP membantu pengembangan program menjadi lebih terstruktur, mudah dipelihara, dan scalable melalui penggunaan class dan object. Dengan konsep encapsulation dan constructor, keamanan serta keteraturan kode lebih terjaga.

3. Class Sederhana (Burung)

Class adalah blueprint atau cetakan untuk membuat objek. Pada contoh ini, class Burung memiliki atribut nama dan warna. Terdapat method terbang() yang menggambarkan perilaku burung. Melalui contoh ini dapat dipahami bagaimana atribut dan method bekerja dalam sebuah class.
