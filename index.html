1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada objek. Objek dibuat berdasarkan class yang memiliki atribut (data) dan method (fungsi). Pendekatan ini membuat program lebih terstruktur, modular, dan mudah dikembangkan.

2. Konsep Dasar OOP

Konsep dasar dalam OOP meliputi: Class, Object, Encapsulation, Inheritance, dan Polymorphism. Konsep-konsep ini membantu dalam membangun sistem yang lebih fleksibel dan reusable.

3. Class Sederhana (Car)

Class adalah blueprint untuk membuat objek. Pada contoh ini dibuat class Car dengan atribut dan method.

4. Object Instantiation

Object instantiation adalah proses membuat objek dari class. Objek dibuat menggunakan constructor.

5. Constructor

Constructor adalah method khusus yang digunakan untuk menginisialisasi nilai awal suatu objek. Constructor dipanggil saat objek dibuat.

6. Practice Class Mahasiswa

Pada praktik ini dibuat class Mahasiswa yang memiliki atribut nama, nim, dan ipk serta method untuk menampilkan data.

7. Encapsulation

Encapsulation adalah teknik menyembunyikan data dengan membuat atribut private dan mengaksesnya melalui getter dan setter. Tujuannya untuk menjaga keamanan data.

8. This Keyword

Keyword this digunakan untuk merujuk pada atribut milik class itu sendiri agar tidak terjadi kesalahan referensi variabel.

9. Product Class

Contoh penggunaan final, required, dan named parameter pada constructor untuk membuat class lebih aman dan fleksibel.

10. Kesimpulan

Dari pembelajaran ini dapat disimpulkan bahwa OOP membantu membuat program lebih terstruktur, mudah dipelihara, dan scalable. Konsep class, object, constructor, encapsulation, dan this keyword memegang peran penting dalam membangun sistem yang baik.

11. Quiz Bonus

Buatlah class Buku dengan atribut judul dan penulis, serta method tampilkan() untuk menampilkan data buku.
