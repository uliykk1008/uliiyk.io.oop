OBJECT ORIENTED PROGRAMMING

Uliya Gumpari | 251410008 | IPK 3,87

1. Pengertian OOP

Object Oriented Programming adalah paradigma pemrograman yang berorientasi pada objek. Pendekatan ini membuat program lebih terstruktur, modular, dan mudah dikembangkan.

2. Konsep Dasar OOP

Konsep utama OOP meliputi Class, Object, Encapsulation, Inheritance, dan Polymorphism. Konsep ini membantu membangun sistem yang fleksibel dan reusable.

3. Class Sederhana (Car)

Contoh pembuatan class dasar.

4. Object Instantiation

Proses membuat objek dari class.

5. Constructor

Constructor digunakan untuk inisialisasi nilai objek.

6. Practice Class Mahasiswa

Implementasi class Mahasiswa beserta method.

7. Encapsulation

Penerapan getter dan setter untuk melindungi data.

8. This Keyword

Penggunaan keyword this untuk referensi atribut class.

9. Product Class

Penggunaan final dan named parameter constructor.

10. Kesimpulan

OOP membantu pengembangan program menjadi lebih terstruktur, mudah dipelihara, dan scalable melalui penggunaan class dan object.

11. Quiz Bonus

Buatlah class Buku dengan atribut judul dan penulis serta method tampilkan().
